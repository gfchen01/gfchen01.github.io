TinySQL - A Simple Database Management System

TinySQL is a simple but fast database management system. TinySQL is self-contained and doesn't require external libraries except standard libraries. TinySQL must be compiled using C++17, GCC 8.0+, CMake 3.0+.

We used some source files of the project: hsql as our SQL Parser. We deeply appreciate their work!

Some important facts

  1. All the files are store in the same directory of the executable tinySQL, under the folder DB_DATAFILES. The file structure:
DB_DATAFILES
| |
INDEX
...(Index files, binary)
| |
RECORDS
...(Table files, binary)
| |
SCHEMAS
...(Schemas of each table, binary)
  1. The database creates an index using B-plus tree on the primary key. Other indeces can be added manually by user.
  2. You can exit tinySQL by input an EOF (ctrl-D on linux), ctrl-c(terminate) easily causes loss of data.

Supported Queries

TinySQL supports the following standard SQL queries:

  1. CREATE TABLE , CREATE INDEX
  2. DROP TABLE, DROP INDEX
  3. INSERT
  4. SELECT (don't support sub-queries, support logical operands like AND)
  5. DELETE (don't support sub-queries, support logical operands like AND)

and the following non-standard SQL queries:

  1. SHOW TABLES; : Display all the tables stored.
  2. INPUT FILE; : Input a text file containing SQL queries. Has to be in uppercase.

supported data types: char (<=20 characters) varchar (<=20 characters) int decimal float

Examples:

create table student(
